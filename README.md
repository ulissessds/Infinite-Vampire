# Infinite-Vampire
Jogo criado como trabalho final para a cadeira de Programação para Jogos I.


Como Jogar
==============
MOVIMENTO:
  \ W, A, S, D
  \ Setas do teclado


ATAQUE:
  \ T
  \ Z
  \ Tab

PULO:
  \ G
  \ Enter
  \ Espaço

SELECIONAR (TITLE SCREEN):
  \ Y
  \ Espaço

VOLTAR (GAME OVER):
  \ H
  \ Esc


========================

*OBS: Também é possível usar um gamepad para jogar
